Opis styl lokalny, liniowy, linijkowy

  1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

    Styl lokalny pozwala na formatowanie tylko jednego, wybranego elementu HTML w dokumencie. Zastosowanie stylu lokalnego pozwala na określenie wyglądu konkretnego elementu, np. paragrafu, nagłówka lub innego tagu HTML, bez wpływania na inne elementy na stronie.

  2. Składnia teoretyczna (z użyciem ramki otaczającej)

    Składnia stylu lokalnego wygląda następująco:

    <style> p { color: red; } </style>

    Aby wyświetlić definicję w ramce, możemy użyć specjalnych znaków:

    <pre>&lt;style&gt; p { color: red; } &lt;/style&gt;</pre>
  3. Co może być selektorem?

    Selektorem w CSS może być tag HTML (np. h1, p), klasa (.nazwa-klasy), identyfikator (#nazwa-id), a także atrybuty elementów. Selektory służą do wskazania, które elementy mają zostać objęte stylem.

  4. Co wpisujemy jako "cechę". Jakie jest znaczenie średników?

    "Cechą" w deklaracji CSS jest właściwość, którą chcemy zmienić, np. color, font-size, background-color. Średnik (;) jest używany do oddzielania poszczególnych deklaracji w jednej regule CSS, co pozwala na uporządkowanie różnych właściwości dla jednego elementu.

  5. Co wpisujemy jako "wartość" (w deklaracji stylu)?

    "Wartość" w deklaracji stylu to wartość przypisana do cechy, np. dla color wartością może być red, #FF0000, lub rgb(255, 0, 0). Wartość ta określa, jak ma wyglądać dana cecha.
