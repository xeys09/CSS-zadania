Wewnętrzny arkusz stylów

  1. Gdzie jest wstawiany?

    Wewnętrzny arkusz stylów jest wstawiany w sekcji <head> dokumentu HTML, wewnątrz znacznika <style>. Znajduje się między otwierającym a zamykającym tagiem <style>.

  2. Składnia teoretyczna (w ramce, konieczne wcięcia)

    Składnia wewnętrznego arkusza stylów wygląda następująco:

    <style>
    body {
        font-family: Arial, sans-serif;
        color: #ddae00;
    }
    p {
        margin: 10px;
        font-size: 14px;
    }
</style>
  3. Jakie mogą być selektory?

    Selektory w wewnętrznym arkuszu stylów mogą być:

    • Tagi HTML (np. h1, p, div),
    • Klasy (.nazwa-klasy),
    • Identyfikatory (#nazwa-id),
    • Grupy selektorów (np. h1, p).
  4. Jakie mogą być cechy i wartości?

    Cechy (właściwości) w CSS to np.:

    • color - kolor tekstu,
    • font-size - rozmiar czcionki,
    • background-color - kolor tła,
    • margin - marginesy,
    • padding - wypełnienie.

    Wartości przypisane do cech to np.:

    • color: red;
    • font-size: 16px;
    • background-color: #ff0000;
    • margin: 10px;
